# s2-token-program

Build a web UI that can:
- handle creation of a token
- it's mint
- transfer of that token
- burning the token supply
- and delegation the ata of this token to another public key.

## Getting Started

### Prerequisites

- Node v18.18.0 or higher

### Installation

#### Clone the repo

```shell
git clone <repo-url>
cd <repo-name>
```

#### Install Dependencies

```shell
npm install
```

#### Start the web app

```
npm run dev
```

## Apps

### web

This is a React app.

#### Commands

Start the web app

```shell
npm run dev
```

Build the web app

```shell
npm run build
```
